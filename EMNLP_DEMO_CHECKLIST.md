# EMNLP Demo Submission Checklist

## Paper

- [ ] Use official EMNLP 2026 style files.
- [ ] Maximum 6 pages for main content.
- [ ] References outside page limit.
- [ ] Optional appendix maximum 2 pages.
- [ ] Include author names and affiliations because Demo Track is single-blind.
- [ ] Replace Mr. Zhang placeholder with full corresponding-author information.
- [ ] Verify all citations and BibTeX entries.
- [ ] Do not include unverified numerical comparisons against DetectGPT, Ghostbuster, Binoculars, or RoBERTa.

## System

- [ ] Public repo accessible.
- [ ] README has Quick Start.
- [ ] Requirements file works in a fresh environment.
- [ ] Web demo or installable package link included in paper.
- [ ] Demo video link included in paper.
- [ ] Example inputs and outputs included.
- [ ] Screenshots included.
- [ ] Evaluation result files or reproduction scripts included.
- [ ] License included.
- [ ] Limitations and ethical-use statement included.
